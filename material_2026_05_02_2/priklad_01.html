Úloha 1: Systematické kódovanie polynómom

Samostatné vypracovanie jednej číslovanej úlohy so zhrnutím použitých definícií.

Zadanie

Zakódujte systematicky pomocou polynómu \(g(x)=x^4+x+1\) informáciu 00000000001010111. Posledný bit napravo je LSB.

Ako to riešiť

Najprv si úlohu rozdeľ na malé rozhodnutia. Pri skúške je dôležitejšie vedieť, ktorý postup použiť, než si pamätať výsledok.

  1. Pri LSB napravo čítaj bity sprava doľava: pravý bit je koeficient pri \(x^0\). Úvodné nuly pri tvorbe polynómu len nič nepridajú.
  2. Pre 00000000001010111 dostaneš \(m(x)=x^6+x^4+x^2+x+1\).
  3. Generujúci polynóm má stupeň \(r=4\), preto systematické kódovanie začína posunom \(x^4m(x)\). Tým necháš miesto pre štyri kontrolné bity.
  4. Vydeľ \(x^4m(x)\) polynómom \(g(x)=x^4+x+1\) a zober iba zvyšok. Nad \(GF(2)\) sa pri delení odčítava XOR-om.
  5. Kódové slovo je \(c(x)=x^4m(x)+r(x)\): vľavo ostane pôvodná informácia a pravé štyri bity sú zvyšok.

Kontrola: Po opätovnom delení \(c(x)\) polynómom \(g(x)\) musí vyjsť nulový zvyšok.

Riešenie krok za krokom

Postup výpočtu

Pravý bit je LSB, takže najprv dostaneme \(m(x)=x^6+x^4+x^2+x+1\). Stupeň \(g(x)\) je \(4\), preto delíme \(x^4m(x)\):

Aktuálny zvyšokPripočítame posunuté \(g(x)\)Nový zvyšok
\(x^10 + x^8 + x^6 + x^5 + x^4\)\(x^{6}\cdot g(x)=x^10 + x^7 + x^6\)\(x^8 + x^7 + x^5 + x^4\)
\(x^8 + x^7 + x^5 + x^4\)\(x^{4}\cdot g(x)=x^8 + x^5 + x^4\)\(x^7\)
\(x^7\)\(x^{3}\cdot g(x)=x^7 + x^4 + x^3\)\(x^4 + x^3\)
\(x^4 + x^3\)\(x^{0}\cdot g(x)=x^4 + x + 1\)\(x^3 + x + 1\)

Posledný zvyšok je \(x^3+x+1\), teda kontrolné bity sú 1011. Po pripojení k pôvodnej správe vznikne systematické slovo.

Výsledok a kontrola

Keďže posledný bit napravo je LSB, informačný polynóm je

\[m(x)=x^6 + x^4 + x^2 + x + 1\]

Stupeň generujúceho polynómu je \(4\), preto pri systematickom kódovaní najprv posunieme správu o štyri miesta:

\[x^4m(x)=x^10 + x^8 + x^6 + x^5 + x^4\]

Vypočítame zvyšok po delení polynómom \(g(x)\):

\[r(x)=\operatorname{rem}(x^4m(x),g(x))=x^3 + x + 1\]

Systematické kódové slovo je \(c(x)=x^4m(x)+r(x)\):

\[c(x)=x^10 + x^8 + x^6 + x^5 + x^4 + x^3 + x + 1\]

Pravé štyri bity 1011 sú zvyšok \(x^3+x+1\); zvyšok vľavo je pôvodná informácia.

V bitoch, stále s LSB napravo, dostaneme

\[\texttt{000000000010101111011}\]
Definície
  • Systematické kódovanie: informačné bity ostanú v kódovom slove viditeľné a doplní sa paritná časť.
  • Zvyšok po delení: polynóm nižšieho stupňa ako generujúci polynóm.
  • LSB napravo: pravý krajný bit je koeficient pri \(x^0\).
