Úloha 3: Mini test na precvičenie

Samostatné vypracovanie jednej číslovanej úlohy so zhrnutím použitých definícií.

Zadanie

Skús si najprv vyriešiť tieto príklady bez pozerania do vypracovania. Sú poskladané podľa typov, ktoré sa opakovali v materiáloch.

  1. Navrhni hardvérové rovnice registra pre \(g(x)=x^4+x+1\).
  2. Vydeľ \(x^9+x^7+x^6+x^3+x\) polynómom \(x^4+x^3+1\).
  3. Systematicky zakóduj správu 1101 pomocou \(g(x)=x^3+x+1\). LSB je napravo.
  4. V \(GF(4)\), kde \(\alpha^2=\alpha+1\), vypočítaj \(\alpha^5/\alpha\).
  5. RS kód nad \(GF(16)\) má opravovať 3 chyby. Koľko koreňov má \(g(x)\), aký je posledný koreň pri prvom koreni \(\alpha^2\), aké sú \(n\), \(k\) a \(|C|\)?
  6. Pre \(g(x)=x^3+x+1\) nájdi syndróm prijatého polynómu \(v(x)=x^5+x^2+1\) a oprav jednobitovú chybu.
Ako to riešiť

Najprv si úlohu rozdeľ na malé rozhodnutia. Pri skúške je dôležitejšie vedieť, ktorý postup použiť, než si pamätať výsledok.

  1. Najprv prepíš zadanie do symbolov a vypíš, čo presne sa má nájsť.
  2. Vyber definíciu alebo vzorec, ktorý zodpovedá typu úlohy.
  3. Počítaj po malých krokoch a každý medzivýsledok redukuj v poli alebo modulo aritmetike, ak to zadanie vyžaduje.
  4. Nakoniec výsledok porovnaj s podmienkou zo zadania.

Kontrola: Výsledok má spĺňať všetky obmedzenia zo zadania, nielen vyzerať algebraicky správne.

Vypracovanie

1. Hardvér pre \(g(x)=x^4+x+1\)

Stupeň je \(4\), takže potrebujeme registre \(R_3,R_2,R_1,R_0\). Nenulové nižšie členy sú pri \(x\) a \(1\):

\[\begin{aligned}R_0'&=b+R_3,\\ R_1'&=R_0+R_3,\\ R_2'&=R_1,\\ R_3'&=R_2.\end{aligned}\]

Prečo práve takto: spätná väzba je najvyšší starý register \(R_3\). Keďže v polynóme sú členy \(x\) a \(1\), \(R_3\) sa XOR-om pripočíta do vstupu pre \(R_1\) a \(R_0\). Pri \(x^2\) a \(x^3\) koeficient \(1\) nie je, preto tam spätnú väzbu nepridávame.

2. Delenie polynómov

\[\frac{x^9 + x^7 + x^6 + x^3 + x}{x^4 + x^3 + 1}\]

Opäť berieme najvyšší člen zvyšku. Posuny, ktoré vzniknú, tvoria podiel:

Aktuálny zvyšokPripočítame posunuté \(g(x)\)Nový zvyšok
\(x^9 + x^7 + x^6 + x^3 + x\)\(x^{5}\cdot g(x)=x^9 + x^8 + x^5\)\(x^8 + x^7 + x^6 + x^5 + x^3 + x\)
\(x^8 + x^7 + x^6 + x^5 + x^3 + x\)\(x^{4}\cdot g(x)=x^8 + x^7 + x^4\)\(x^6 + x^5 + x^4 + x^3 + x\)
\(x^6 + x^5 + x^4 + x^3 + x\)\(x^{2}\cdot g(x)=x^6 + x^5 + x^2\)\(x^4 + x^3 + x^2 + x\)
\(x^4 + x^3 + x^2 + x\)\(x^{0}\cdot g(x)=x^4 + x^3 + 1\)\(x^2 + x + 1\)

Preto vyjde

\[q(x)=x^5 + x^4 + x^2 + 1,\qquad r(x)=x^2 + x + 1\]

3. Systematické kódovanie

Správa 1101 znamená \(m(x)=x^3+x^2+1\). Stupeň \(g(x)\) je \(3\), preto počítame zvyšok z \(x^3m(x)\):

\[x^3m(x)=x^6+x^5+x^3\]
Aktuálny zvyšokPripočítame posunuté \(g(x)\)Nový zvyšok
\(x^6 + x^5 + x^3\)\(x^{3}\cdot g(x)=x^6 + x^4 + x^3\)\(x^5 + x^4\)
\(x^5 + x^4\)\(x^{2}\cdot g(x)=x^5 + x^3 + x^2\)\(x^4 + x^3 + x^2\)
\(x^4 + x^3 + x^2\)\(x^{1}\cdot g(x)=x^4 + x^2 + x\)\(x^3 + x\)
\(x^3 + x\)\(x^{0}\cdot g(x)=x^3 + x + 1\)\(1\)

Systematické kódové slovo je posunutá správa plus zvyšok. Nad \(GF(2)\) je plus XOR, takže zvyšok iba doplní kontrolné bity.

\[r(x)=1\]
\[c(x)=x^6 + x^5 + x^3 + 1\]
\[\texttt{1101001}\]

4. Počítanie v \(GF(4)\)

Z rovnosti \(\alpha^2=\alpha+1\) dostaneme

\[\alpha^3=\alpha(\alpha+1)=\alpha^2+\alpha=(\alpha+1)+\alpha=1\]

Nenulové prvky sa teda opakujú s periódou \(3\). Pri delení mocnín odčítame exponenty:

\[\alpha^5/\alpha=\alpha^4=\alpha\]

Použili sme \(\alpha^3=1\), takže \(\alpha^4=\alpha^3\alpha=\alpha\).

5. RS kód nad \(GF(16)\)

Pre Reed-Solomonov kód stačí v týchto príkladoch tento recept: na opravu \(t\) chýb potrebujeme \(2t\) po sebe idúcich koreňov. Stupeň generujúceho polynómu je teda \(2t\).

\[t=3\Rightarrow 2t=6\text{ koreňov}\]
\[\alpha^2,\alpha^3,\alpha^4,\alpha^5,\alpha^6,\alpha^7\]

Prvý koreň je \(\alpha^2\). Keď máme šesť koreňov, posledný exponent je \(2+5=7\). Pre plný RS kód nad \(GF(q)\) je \(n=q-1\), teda pri \(q=16\) platí:

\[n=16-1=15,\qquad k=15-6=9,\qquad |C|=16^9\]

6. Syndróm a oprava

\[s(x)=\operatorname{rem}(x^5+x^2+1, x^3+x+1)=x\]

Aby sme vedeli, ktorá jednobitová chyba má tento syndróm, porovnáme ho so zvyškami pre \(1,x,x^2,\ldots,x^6\):

\(x\)\(x\)jednobitová chyba na pozícii 1
\(x^2\)\(x^2\)jednobitová chyba na pozícii 2
\(x + 1\)\(x^3\)jednobitová chyba na pozícii 3

Syndróm \(x\) zodpovedá jednobitovej chybe \(e(x)=x\). Opravíme:

\[c(x)=v(x)+e(x)=x^5 + x^2 + x + 1\]
