Úloha 2: Delenie polynómov nad GF(2)

Samostatné vypracovanie jednej číslovanej úlohy so zhrnutím použitých definícií.

Zadanie

Vydeľte nad \(GF(2)\)

\[\frac{x^7 + x^5 + x^4 + x^2 + 1}{x^4 + x + 1}\]

Určte podiel aj zvyšok.

Ako to riešiť

Najprv si úlohu rozdeľ na malé rozhodnutia. Pri skúške je dôležitejšie vedieť, ktorý postup použiť, než si pamätať výsledok.

  1. Pracuj nad \(GF(2)\), takže rovnaké členy sa rušia: \(x^i+x^i=0\).
  2. V každom kroku porovnaj najvyšší člen aktuálneho zvyšku s najvyšším členom deliteľa.
  3. Rozdiel stupňov dá člen podielu; týmto členom vynásob deliteľ.
  4. Výsledok pripočítaj XOR-om k aktuálnemu zvyšku a pokračuj, kým stupeň zvyšku nebude menší ako stupeň deliteľa.

Kontrola: Výsledok musí spĺňať \(f(x)=q(x)g(x)+r(x)\) a zvyšok musí mať menší stupeň ako deliteľ.

Vypracovanie

Pri delení nad \(GF(2)\) neodčítavame klasicky. Odčítanie je rovnaké ako sčítanie, teda rovnaké členy sa vyrušia: \(x^i+x^i=0\).

Postup je vždy rovnaký: pozrieme sa na najvyšší člen aktuálneho zvyšku, porovnáme ho s najvyšším členom deliteľa a podľa rozdielu stupňov určíme posun. Tento posun si zapamätáme do podielu.

V prvom kroku je najvyšší člen \(x^7\) a deliteľ má najvyšší člen \(x^4\). Preto použijeme posun \(x^{7-4}=x^3\), teda do podielu pribudne \(x^3\). Potom XOR-om pripočítame \(x^3g(x)\). Rovnaký postup opakujeme, kým zvyšok nemá stupeň menší ako \(4\).

Aktuálny zvyšokPripočítame posunuté \(g(x)\)Nový zvyšok
\(x^7 + x^5 + x^4 + x^2 + 1\)\(x^{3}\cdot g(x)=x^7 + x^4 + x^3\)\(x^5 + x^3 + x^2 + 1\)
\(x^5 + x^3 + x^2 + 1\)\(x^{1}\cdot g(x)=x^5 + x^2 + x\)\(x^3 + x + 1\)

Posuny v tabuľke boli \(x^3\) a \(x\), preto je podiel \(x^3+x\). Posledný zvyšok má stupeň \(3\), čo je menej ako stupeň deliteľa \(4\), takže delenie končí.

\[q(x)=x^3 + x\]
\[r(x)=x^3 + x + 1\]

Výsledok teda je

\[x^7 + x^5 + x^4 + x^2 + 1=(x^3 + x)(x^4 + x + 1)+(x^3 + x + 1)\]
Definície
  • Delenie v \(GF(2)\): vyberieme najvyšší člen, posunieme deliteľ a XOR-om ho odčítame.
  • Zvyšok: musí mať menší stupeň ako deliteľ.
  • Kontrola výsledku: \(f(x)=q(x)g(x)+r(x)\).
